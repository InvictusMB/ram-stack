Usage: composition-root build <rule> <search-root> [options]

  Generates composition root files in a "search-root" folder for a "rule" from composition configuration file.

Options:

  -c <file>, --config <file>      composition config file (default: "composition-config.json").
  -o <file>, --output <file>     output file
  -h, --help                      Display this usage info.
  -v, --version                   Display the cpr version.
